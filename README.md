# Koro Felix — Resmi Web Sitesi

Koro Felix'in resmi tanıtım web sitesi. React, Tailwind CSS ve Vite ile geliştirilmiştir.

## Proje Hakkında

Bu proje, Ankara merkezli müzik platformu Koro Felix için sıfırdan tasarlanmış ve geliştirilmiş tek sayfalık (SPA) bir landing sitesidir. Çocuk korolarından orkestralara, pop'tan klasiğe 8 farklı müzik grubunu tanıtmaktadır.

## Teknoloji Yığını

- **React 18** — UI bileşenleri
- **Vite** — geliştirme sunucusu ve build aracı
- **Tailwind CSS** — utility-first CSS framework
- **Framer Motion** — sayfa geçişi ve scroll animasyonları
- **react-intersection-observer** — görünürlük tetiklemeli animasyonlar

## Özellikler

- Tam responsive tasarım (mobil, tablet, masaüstü)
- Scroll-triggered animasyonlar (framer-motion + useInView)
- Mobil hamburger menü (drawer animasyonu)
- Masonry galeri (CSS columns)
- Grayscale → renkli hover efekti (ekip fotoğrafları)
- Google Fonts entegrasyonu (Playfair Display + DM Sans)
- Smooth scroll navigasyon

## Bölümler

| Bölüm | Açıklama |
|---|---|
| Hero | Ana başlık, istatistikler, dekoratif halkalar |
| Hakkımızda | Vizyon/Misyon, çalışma alanları, alıntı |
| Korolarımız | 8 koro/grup kartı, hover detayları |
| Ekibimiz | 8 sanatçı kartı, biyografi overlay |
| Galeri | Konser ve sahne arkası fotoğrafları |
| İletişim | Adres, iletişim formu, sosyal medya |

## Kurulum

```bash
npm install
npm run dev
```

Uygulama `http://localhost:5173` adresinde çalışır.

## Build

```bash
npm run build
```
